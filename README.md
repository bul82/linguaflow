# LinguaFlow — Языковая школа

> Лендинг для языковой школы с интерактивными элементами: выбор языка и уровня, тест на определение уровня, карточки преподавателей, расписание, тарифы и форма записи на пробный урок.

## Стек

- **HTML5** — семантическая разметка
- **CSS3** — кастомные свойства, flexbox, grid, анимации
- **Vanilla JavaScript (ES6+)** — интерактивность без фреймворков
- **Google Fonts** — Inter
- **Lucide Icons** — иконки через CDN

## Секции

| Секция | Описание |
|--------|----------|
| Hero | Заголовок, описание, CTA-кнопки, декоративные языковые пузыри |
| Языки и уровни | Tabs для выбора языка (EN/ES/DE/FR/ZH), карточки уровней A1-C2 |
| Тест на уровень | 5 вопросов с вариантами ответов, прогресс-бар, результат с рекомендацией |
| Преподаватели | Карточки с фото, специализацией, рейтингом, стажем |
| Расписание | Таблица с фильтром онлайн/офлайн |
| Форматы | Карточки с преимуществами онлайн и офлайн обучения |
| Тарифы | 3 тарифа с переключением месяц/год |
| Пробный урок | Форма записи с валидацией |
| Личный кабинет | Модальное окно входа/регистрации |

## Адаптивность

- **Desktop** (> 1024px) — полный layout
- **Tablet** (768-1024px) — 2 колонки
- **Mobile** (< 768px) — single column, hamburger menu

## Запуск

Просто откройте `index.html` в браузере — никаких зависимостей и сборки не требуется.

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

- Single-file архитектура — легко передать клиенту
